====== git ======
Created 목요일 14 6월 2018

git 을 로컬 저장소와 원격 저장소가 필요하며, 이를 사용하기 위해서는 노트와 git hub 가필요합니다.
 git은 텍스트 형태에 정보를 저장하고 관리를 할수 있습니다.


프로그램을 다 설치 했다면 시작에서
 git bash를 열어 텍스트가 있는 폴더까지 갑니다.
거기서 파일이 있는지 확인을 하시고 있으면 스테이징과 커밋을 해야됩니다. 스테이징은 인덱스 목록으로 저장하는 단계이며 커밋을 해야 
원격 저장소에 저장이 가능합니다. 커밋을 한후에 원격에 저장해보세요~

처음 사용자는 처음하는 과정을 따라 하시고
처음할때는 사용자 정보가 필요합니다~~
계정이 있는사람은 추가하는 작업을 하세요~
